# DrumKit Player


### ⚙ How to run it
- Run "playDrumkit (#NewWave).py"
- Choose your Drum Kit directory 

### The story of the code
Cela fait plusieurs année que je n'ai pas vraiment touché à mon logiciel de composition (FL). J'étais trop chaud...<br>
Mon atout par rapport au autres je pense c'était le fait que je connaissais bien mon drumkit, mes instruments et donc 
dès que j'avais un truc dans la tête, une inspi - bim - je savais quoi chercher et où le trouver.<br>
J'ai donc créer ce programme qui joue en boucle (aléatoirement ou non) chaque drum de mon dossier de drums persos. J'ai ça dans mes écouteurs dans mes trajet de bus comme un fou ptdr.
L'objectif est de retrouver cette familiarité que j'avais avec mes drums perso.
